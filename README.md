
# 📊 Amazon Sales Dashboard | Power BI

## 📌 Project Overview

The **Amazon Sales Dashboard** is an interactive Business Intelligence dashboard built using **Power BI** to analyze Amazon sales performance. It provides a clear overview of key business metrics, sales trends, regional performance, and category-wise insights to support data-driven decision-making.

## 🚀 Features

* Executive Sales Overview
* Interactive KPI Cards
* Monthly Sales Trend Analysis
* Top 10 Sub-Categories by Sales
* Profit Analysis by Category
* Regional Sales Performance
* Sales Distribution by Category
* Interactive Filters (Region & Category)
* Reset Filters Button

## 🛠️ Tools & Technologies

* Power BI
* Power Query
* DAX
* Data Modeling
* Data Cleaning & Transformation

## 📈 Dashboard KPIs

* Total Sales
* Total Profit
* Total Orders
* Total Customers
* Total Quantity
* Profit Margin (%)

## 📊 Key Insights

* Technology is the highest-performing category by sales.
* The West region contributes the highest sales.
* Monthly sales show noticeable seasonal trends.
* A small number of sub-categories generate a large share of total revenue.
